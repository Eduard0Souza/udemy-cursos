O elemento HTML ol é usado para criar listas ordenadas, onde a ordem dos itens é significativa. O atributo type do elemento ol permite especificar o tipo de numeração ou marcadores usados para identificar cada item na lista.

Exemplos de tipos de numeração:

  1. type="1": Números arábicos (1, 2, 3, ...).
  2. type="A": Letras maiúsculas (A, B, C, ...).
  3. type="a": Letras minúsculas (a, b, c, ...).
  4. type="I": Números romanos maiúsculos (I, II, III, ...).
  5. type="i": Números romanos minúsculos (i, ii, iii, ...).
