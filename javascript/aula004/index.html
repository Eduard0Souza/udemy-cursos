Array em JavaScript

Em JavaScript, um array é uma estrutura de dados que armazena uma coleção ordenada de valores, podendo ser de diferentes tipos, acessados por um índice. Arrays são utilizados para agrupar e manipular múltiplos dados de uma só vez, facilitando o desenvolvimento de aplicações.

Como declarar e usar arrays:

Declaração: Arrays podem ser declarados usando colchetes [] ou a função new Array(). Acesso: Os elementos do array são acessados por meio de seu índice (posição), começando em 0. Manipulação: JavaScript oferece diversos métodos para manipular arrays, como adicionar, remover, modificar e iterar sobre seus elementos.

Métodos de array:

A linguagem possui diversos métodos nativos para trabalhar com arrays, como push, pop, shift, unshift, splice, slice, map, filter, reduce, entre outros.

Exemplo:

let frutas = ["maça", "banana", "laranja"]
