CSS- Cascading Style Sheets

Cascading Style Sheets (Folhas de estilo em cascata) é uma linguagem usada para definir o estilo e a formatação de documentos web, como páginas HTML. Em resumo, o CSS controla a aparência visual de um site, incluindo cores, fontes, espaçamento e layout, separando-o da estrutura (HTML).

O que o CSS faz?

Define a apresentação: O CSS determina como os elementos HTML serão exibidos na tela, incluindo cores, fontes, tamanhos, espaçamento, posicionamento e outros estilos visuais.

Separa conteúdo e apresentação: O CSS permite que você separe o conteúdo de uma página (HTML) da sua apresentação visual, facilitando a manutenção e atualização do site.

Cria designs responsivos: Com o uso de media queries, o CSS pode adaptar o layout do site para diferentes tamanhos de tela e dispositivos, tornando-o responsivo.

Como funciona o CSS? Regras de estilo: O CSS utiliza regras para definir como os elementos HTML devem ser estilizados. Cada regra consiste em um seletor (que identifica os elementos a serem estilizados) e um bloco de declarações (que especificam as propriedades de estilo).

Seletor: O seletor identifica o elemento HTML ao qual a regra se aplica. Por exemplo, h1 seleciona todos os elementos de cabeçalho de nível 1.

Declarações: As declarações definem as propriedades de estilo e seus valores. Por exemplo, color: blue; define a cor do texto para azul, e font-size: 20px; define o tamanho da fonte para 20 pixels.

Cascata: O termo "cascata" no nome CSS refere-se à forma como as regras de estilo são aplicadas. Regras mais específicas ou definidas mais tarde no código podem substituir regras mais gerais ou definidas anteriormente.
