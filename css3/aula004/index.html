DIV's e SPAN's

Em HTML, div e span são elementos genéricos usados para estruturar e estilizar conteúdo, mas com diferenças cruciais em como são exibidos e usados.

DIVs (Divisão)

Elemento de nível de bloco:
Div ocupa toda a largura disponível e cria uma quebra de linha antes e depois do seu conteúdo.

Estrutura:
É usado para agrupar grandes blocos de conteúdo, como seções inteiras de uma página, e para organizar elementos na página.

Semântica: Div por si só não tem significado semântico, ou seja, não representa um tipo específico de conteúdo. É usado principalmente como um contêiner para fins de estilo e layout

Spans

Elemento inline:
Span não cria quebra de linha e se comporta como parte do texto ou de outro elemento inline

Estilização
É usado para aplicar estilos específicos a partes menores de conteúdo, como por exemplo um trecho de texto, uma palavra ou um ícone dentro de um parágrafo.

Semântica
Span também não tem significado semântico e é usado principalmente para fins de estilização de conteúdo com CSS ou Javascript.

Principais Diferenças

Exibição: Div é um elemento de bloco, enquanto Span é um elemento inline.
Escopo: Div é usado para agrupar grandes seções, enquanto Span é usado para partes menores de conteúdo.
