Classes e IDs

Em CSS, id e class são atributos usados para identificar e estilizar elementos HTML, mas com propósitos diferentes. ID é usado para identificar um único elemento na página, enquanto class pode ser usado para identificar múltiplos elementos.

Diferenças

ID:
  • Identifica um elemento único na página.
  • É usado quando você precisa aplicar um estilo específico a um único elemento.
  • É referenciado no CSS com o símbolo #.
  • Exemplo: div id="meu-id"...div e #meu-id { color: blue; }.

CLASS:
  • Identifica um grupo de elementos.
  • É usado quando você quer aplicar o mesmo estilo a vários elementos.
  • É referenciado no CSS com o símbolo ..
  • Exemplo: div class="minha-classe".../div e p class="minha-classe">... e .minha-classe { color: red; }

Quando usar cada um

ID
Use id quando você precisa identificar um elemento único na página, como um cabeçalho, rodapé, ou um elemento com funcionalidade específica que não se repete.

CLASS
Use class quando você precisa aplicar o mesmo estilo a múltiplos elementos, como botões com o mesmo design, elementos de uma lista, ou parágrafos com formatação similar.
